New Cluster

Cluster name is required.

Cluster name should have Minimum 3 characters.

Cluster name should have Maximum 64 characters.

Cluster name can only include alphanumeric characters including space and dash.

Cluster key is required.

Key must be minimum 3 characters long.

Key should be maximum 32 characters long.

Only lowercase alphanumeric characters are allowed

A valid Management IP address is required

Provisioning State is required.

REST URI is required.
